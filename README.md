# 3QuickMaths

Android math quiz app.

## Features

- Practice Mode
- Infinity Mode
- Multiple difficulty levels
- Progress tracking

## Topics

- Basic arithmetic
- Exponents
- Fractions
- Percentages
- Order of operations

## Stack

- Kotlin
- Jetpack Compose
- Material Design 3

## Setup

```
git clone https://github.com/YOUR_USERNAME/3quickmaths.git
```

Open in Android Studio. Build. Run.
