# 3QuickMaths

A fast-paced math quiz game for Android that helps sharpen your Calculus skills.

## Features

- **Practice Mode** — Learn at your own pace with unlimited attempts per topic
- **Infinity Mode** — Race against the clock with endless questions from multiple topics
- **Multiple Difficulty Levels** — Easy, Medium, and Hard questions to match your skill level
- **Progress Tracking** — Track your stats and see your improvement over time
- **Sound & Haptic Feedback** — Satisfying audio and vibration cues for correct/incorrect answers

## Topics Covered

- Addition & Subtraction
- Multiplication & Division
- Exponents & Square Roots
- Fractions & Decimals
- Percentages
- Order of Operations
- And more!

## Built With

- **Kotlin** — Modern, concise Android development
- **Jetpack Compose** — Declarative UI toolkit
- **Material Design 3** — Clean, modern interface

## Getting Started

1. Clone the repository
2. Open in Android Studio
3. Build and run on your device or emulator
